# Password Generator

A simple web application to generate strong, random passwords with customizable options. Built using **HTML, CSS, and JavaScript**.

## Features
- Generates strong and random passwords
- Options to select password length
- Include/exclude uppercase, lowercase, numbers, and symbols
- One-click copy to clipboard
- Clean black & green user interface

## Usage
1. Open `index.html` in your browser
2. Select options for password generation
3. Click **Generate**
4. Click **Copy** to copy the password to clipboard
